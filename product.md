# Product Overview

Weather Lookup is a standalone browser-based weather application that provides real-time weather information for any city worldwide. The app features a clean, modern interface with responsive design and runs entirely in the browser without requiring a server.

## Key Features

- Real-time weather data via OpenWeatherMap API
- City-based weather search
- Display of temperature, conditions, humidity, and wind speed
- Demo mode for testing without API key
- Mobile-responsive design
- Client-side validation and error handling
